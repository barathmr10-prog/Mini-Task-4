# Task-4---Flask-Rest-API
I have created a empty dictionary and created a index variable for key and user details as values .

I used FLASK to create app, JSONIFY and request methods to create, read, update and delete the user data.

In the api endpoint = http://127.0.0.1:5000 :-
I used "POST" in POSTMAN and get the data from request and added it to my dictionary
I used GET to view all of my user detail

In the api endpoint = http://127.0.0.1:5000/pk :-
I used PUT, DELETE to update and Delete the user data with its PK id, which is key of the dict.
